活动断层采集子系统

  • 时间：2020年11月至2020年12月
  • 项目简介：活动断层采集子系统是给中国国家地震局做的一款内部使用的APP，用于帮助外部作业人员采集活动断层的信息并上传到服务器。

  • 技术点：

    1、ArcGIS SDK加载天地图瓦片图层

    2、使用ArcGIS在图层上进行绘图

  • 架构

    1、语言：Objective-C

    2、网络请求：AFNetworking

    3、UI:Storyboard

    4、数据库：sqlite + BGFMDB

    5、地图服务：ArcGIS + 天地图图层
