活动断层采集子系统（2020年11月-2020年12月）

  • 时间

    2020年11月至2020年12月

  • 项目简介

    活动断层采集子系统是给中国国家地震局做的一款内部使用的APP，用于帮助外部作业人员采集活动断层的信息并上传到服务器。项目采用了ArcGIS和天地图来作为地图服务，因为考虑到采集地可能在山区，无网络环境下使用了sqlite进行数据存储。

  • 技术点

    1、ArcGIS SDK加载天地图瓦片图层

    2、使用ArcGIS在图层上进行绘图

  • 架构

    1、语言：Objective-C

    2、网络请求：AFNetworking

    3、UI:Storyboard + BDToolKit + ZXToolbox

    4、数据库：sqlite + BGFMDB

    5、地图服务：ArcGIS SDK + 天地图图层

    6、其他：YYKit SDWebImage IQKeyboardManager MJRefresh SVProgressHUD Reachability等

铺小满（2020年08月-2020年10月）

  • 时间

    2020年08月至2020年10月

  • 项目简介

    铺小满是一款本地化工具类型的APP,主要为商铺提供直播带货，店铺管理，店铺分销，会员返利，会员裂变，增加优惠券，大转盘抽奖，商分销，拼团，秒杀等功能来帮助商户锁定私域流量。

  • 技术点

    1、使用腾讯直播SDK实现推拉流、直播

    2、使用websocket实现聊天系统

    3、支持微信登陆、微信支付、支付宝支付

  • 架构

    1、语言：Objective-C

    2、网络请求：AFNetworking

    3、UI:Storyboard + BDToolKit + ZXToolbox

    4、直播：腾讯直播SDK

    5、聊天系统：WebSocket + SocketRocket

    DSIM（2020年02月-2020年06月）

    • 时间

      2020年02月至2020年06月

    • 项目简介

      DSIM是一款以SIM币为基础的分销小游戏。用户通过平台号卡充值获得SIM币，每天会返一定数量的币到平台钱包。SIM币可以用于玩平台游戏，比如夺宝和H5小游戏。在夺宝和H5小游戏里，用户可以赢取更多的SIM币。平台还会设置一些奖励机制，比如空投SIM币和分红SIM币。用户如果推荐了好友注册DSIM并通过了号卡充值，就可以获得节点分红。SIM币在支点交易所上市交易，还可以通过OTC的方式进行交易流通。

    • 技术点

      1、采用SVGA动画图形的解析实现游戏动效

      2、其他动画效果的实现

    • 架构

      1、语言：Objective-C

      2、网络请求：AFNetworking

      3、UI:Storyboard + BDToolKit + ZXToolbox
