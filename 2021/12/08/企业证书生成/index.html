企业证书生成

Author: 班殿宏 Date: 2021年12月08日  11:05:32

本文只介绍发布证书的生成，开发证书生成类似。

1、Certificates

New -> In-House and Ad Hoc -> Choose File -> Done

2、Identifiers

New -> App IDs -> Register an App ID -> Done

注意，Deployment Details根据情况勾选。如图：

3、Profiles

New -> In House -> Select an App ID -> Select Certificates -> Review, Name and Generate -> Done

另：VPN签名证书生成类似，但不可使用发布证书做VPN签名。
